About the Project

Cigar-Vault is a full-stack application built with React, Express.js, Socket.IO, and PostgreSQL. It allows users to log, rate, and track their cigar experiences in a clean, scalable interface.

Key Features

  • User authentication & profile management
  • Real-time updates using Socket.IO
  • Database storage with PostgreSQL
  • Responsive design for mobile and desktop

Tech Stack

React Express.js Socket.IO PostgreSQL
